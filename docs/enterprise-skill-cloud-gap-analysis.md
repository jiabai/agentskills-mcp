# 企业私有云需求与当前实现差异清单

**文档目的**：对照《企业级私有云技能服务需求文档》与当前代码实现，列出差异与待实现功能，作为落地评估清单。

---

## 已实现或部分覆盖

- **用户与登录**：已实现邮箱验证码登录/注册/刷新（JWT）。
- **Skill 基础管理**：已实现 Skill CRUD、文件上传、文件列表与读取，但仅文件级上传，不含 ZIP/版本能力。
- **MCP 服务入口**：已提供 `/mcp` 与 `/sse` 入口、健康检查与基础指标。
- **MCP 认证**：当前采用 API Token（`ask_live_`）认证，具备用户隔离上下文注入。
- **运营统计**：已实现概览指标聚合与清理/清零接口，不等同审计日志体系。

---

## 核心差异点

### 1) 权限与企业认证
- **需求**：LDAP/AD、SSO、MFA、RBAC、企业/团队/个人可见性。
- **现状**：仅基础用户与 JWT 认证，Skill 模型无可见性与权限字段。
- **差异**：缺少企业认证接入、权限模型与审计链路。

### 2) Skill 生命周期管理
- **需求**：ZIP 上传、metadata 解析、版本管理/回滚、下架策略、依赖管理。
- **现状**：单文件上传到 Skill 目录，不解析 metadata，不支持版本与回滚。
- **差异**：缺少版本体系、依赖解析、下架策略与 ZIP 处理。

### 3) MCP/接口契约
- **需求**：`skill://list`、`skill://{id}@{version}`、`execute_skill` 工具规范与技能加密下载接口。
- **现状**：FlowLLM 基础工具加载逻辑，无 `skills/download` 接口。
- **差异**：缺少标准化资源接口与技能下载协议。

### 4) 缓存与加密
- **需求**：加密传输与存储、KMS、客户端本地缓存、过期清理与离线降级。
- **现状**：技能文件本地明文存储，无 KMS 与缓存策略实现。
- **差异**：缺少加密与缓存生命周期体系。

### 5) 审计与合规
- **需求**：审计日志采集、不可篡改存储、查询与导出。
- **现状**：仅运营指标统计与日志输出。
- **差异**：缺少审计模型、采集点与导出接口。

### 6) 存储与部署架构
- **需求**：对象存储（MinIO/S3）承载技能文件。
- **现状**：使用本地文件系统。
- **差异**：缺少对象存储适配与生命周期管理。

## 需要补齐的功能清单（按优先级）

### P0
- LDAP/AD/SSO 接入与用户身份映射
- RBAC 与可见性模型（企业/团队/个人）
- ZIP 上传、metadata 解析与入库
- Skill 版本体系与回滚
- Skill 下载接口与加密传输
- 审计日志采集、查询与导出

### P1
- 依赖管理（声明、解析、安装策略）
- 本地缓存与过期清理、离线降级策略
- 对象存储（MinIO/S3）接入

### P2
- MFA 多因素认证
- 审计日志不可篡改存储策略
