# 企业私有云需求与当前实现差异清单

**文档目的**：对照《企业级私有云技能服务需求文档》与当前代码实现，列出差异与待实现功能，作为落地评估清单。

---

## 已实现或部分覆盖

- **用户与登录**：已实现邮箱验证码登录/注册/刷新（JWT）。
- **Skill 基础管理**：已实现 Skill CRUD、文件上传、文件列表与读取，并已补齐 ZIP 上传、版本与回滚、下架与缓存失效标记、版本差异对比、依赖安装指引。
- **MCP 服务入口**：已提供 `/mcp` 与 `/sse` 入口、健康检查与基础指标。
- **MCP 认证**：当前采用 API Token（`ask_live_`）认证，具备用户隔离上下文注入。
- **运营统计**：已实现概览指标聚合与清理/清零接口，不等同审计日志体系。

---

## 核心差异点

### 1) 权限与企业认证
- **需求**：LDAP/AD、SSO、MFA、RBAC、企业/团队/个人可见性。
- **现状**：邮箱验证码登录 + JWT、MCP API Token；用户仅 `is_superuser` 管理标记；Skill 仅以 `user_id` 归属，无可见性/角色字段。
- **差异**：缺少 LDAP/AD/SSO/MFA 接入、企业身份映射与组织/团队模型；缺少 RBAC 与可见性控制；无权限变更审计链路。

### 2) Skill 生命周期管理
- **需求**：ZIP 上传、metadata 解析、版本管理/回滚、下架策略、依赖管理。
- **现状**：已支持 ZIP 上传与 `SKILL.md` frontmatter 解析；具备版本归档、版本列表、回滚；具备下架/启用与 `cache_revoked_at` 缓存失效标记；API 文件读取与 MCP 工具在下架后会返回 `SKILL_DEACTIVATED`；依赖信息存于版本记录并提供安装指引与版本差异对比接口。
- **差异**：版本自动递增未实现；技能搜索仅支持 name/description，缺少标签维度；依赖管理仍为客户端安装策略，缺少统一声明规范与多语言生态适配。

### 3) MCP/接口契约
- **需求**：`skill://list`、`skill://{id}@{version}`、`execute_skill` 工具规范与技能加密下载接口。
- **现状**：MCP 仅提供 `load_skill_metadata` / `load_skill` / `read_reference_file` / `run_shell_command` 工具；无 `skill://` 资源接口与 `execute_skill`；无 `skills/download` 接口。
- **差异**：缺少标准化资源接口、技能执行工具契约与加密下载协议；MCP 认证方式为 API Token，与需求 JWT 认证不一致。

### 4) 缓存与加密
- **需求**：加密传输与存储、KMS、客户端本地缓存、过期清理与离线降级。
- **现状**：技能文件存放在本地文件系统（`SKILL_STORAGE_PATH`），无加密存储、无 KMS、无本地缓存/过期清理/离线降级实现。
- **差异**：缺少端到端加密与缓存生命周期体系。

### 5) 审计与合规
- **需求**：审计日志采集、不可篡改存储、查询与导出。
- **现状**：仅有调用统计聚合（`request_metrics`）与基础访问日志输出。
- **差异**：缺少审计模型、采集点、不可篡改存储与查询/导出接口。

### 6) 存储与部署架构
- **需求**：对象存储（MinIO/S3）承载技能文件。
- **现状**：使用本地文件系统存储技能文件。
- **差异**：缺少对象存储适配与生命周期管理。

## 需要补齐的功能清单（按优先级）

### P0
- LDAP/AD/SSO 接入与用户身份映射
- RBAC 与可见性模型（企业/团队/个人）
- 技能版本自动递增
- MCP 接口契约（`skill://list` / `skill://{id}@{version}` / `execute_skill`）
- Skill 下载接口与加密传输
- 审计日志采集、查询与导出

### P1
- 技能搜索（标签维度）
- 依赖管理（声明规范、多语言适配、安装策略）
- 本地缓存与过期清理、离线降级策略
- 对象存储（MinIO/S3）接入

### P2
- MFA 多因素认证
- 审计日志不可篡改存储策略
